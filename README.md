# VideoBot


VideoBot is a web-based application that helps creators generate, manage, and publish short-form videos using AI-powered tools.


## Features
- AI-generated short-form videos
- Automated caption and prompt handling
- Optional scheduling and posting to supported platforms
- Simple web-based interface


## How It Works
Users create or upload content, customize prompts, and generate short videos optimized for social platforms like TikTok. The app handles rendering and organization while users stay in control of publishing.


## Privacy
VideoBot does not collect personal data beyond what is required to provide its services. User authentication and platform permissions are handled securely.
